## Overview

This a Tic Tac Toe game made in HTML, CSS and vanilla JS. It has the game and some more.

The app is in the Spanish language for ease when coding.

### Build with:

- **HTML5** - the main page.
- **CSS3** - the styles.
- **Java Script** - the logic.

## Usage

You can access the game by the [deployed website](https://tic-tac-toe-js-abc.netlify.app/);

You can play the game normally just by pressing the spaces that are allowed.

## Features

- **Fully playable game**.
- The **game is locked** after the game ends.
- The game can end in a **tie**.
- You can **change the names** of the players.
- **Reset the game** anytime by pressing a button.
- **Diferrent style for small width** screens (like vertical phones).
  
## To Take In Consideration

- The application main language is spanish, so every error message will be spanish. There's no english.
